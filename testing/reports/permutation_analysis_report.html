Model Permutation Analysis Report

Analysis of 36 Bedrock model permutations (9 vision x 4 embedding) testing the same input (90 personas, 5 Likert questions). Goal: identify which permutations agree, which are outliers, and the best-value consensus model (cheapest and closest to centroid).

How the Composite Score is calculated

Each experiment runs 90 AI personas through 5 Likert-scale questions (scored 1-5). The composite score is a weighted sum that combines all 5 question results into a single 0-1 number:

  1. For each question, compute the mean score across all 90 personas (range 1-5).
  2. Normalize each mean to a 0-1 scale: (mean - 1) / 4   so Likert 1 = 0, Likert 3 = 0.5, Likert 5 = 1.0.
  3. Weight and sum: multiply each normalized score by its question weight, then sum: composite = sum(normalized_i x weight_i)

The result is a value between 0 and 1, where higher = stronger agreement with the tested concept. Weights are defined per survey and sum to 1.0, so each question's influence reflects its importance to the overall assessment.

S7. Key Findings & Recommendation

Findings

  1. 33 of 36 permutations (91%) produce consistent results within the consensus cluster.
  2. 3 outlier permutations identified, involving: Nova 2 Lite, Sonnet 3.
  3. Vision model choice drives 1.8x more variation than embedding model choice.
  4. Most consistent vision model (across embeddings): Sonnet 4.5 (std dev 0.0218).
  5. Least consistent vision model: Sonnet 3 (std dev 0.0960).
  6. Most consistent embedding model: Titan v1 (std dev 0.0332).
  7. Best value model: Nova 2 Lite + Titan v2 ($0.08, centroid dist 0.378) - 99% cheaper than Opus 4.5 + Titan v2 ($9.68).
  8. Composite score range: 0.438 to 0.567 (median 0.5025, std dev 0.0648).
  9. Amazon Nova models: 11 of 12 permutations are in the consensus cluster, validating them as cost-effective alternatives.

Recommendation

Use Nova 2 Lite + Titan v2 at $0.08/experiment (centroid distance: 0.378).

This model offers the best balance of low cost and closeness to the group consensus centroid (including premium models like Opus 4.5 and Sonnet 4.5), delivering 99% cost savings compared to the most expensive consensus option ($9.68).
